Ονοματεπώνυμο: Στέφανος Λάης
============================

Το load limit λ ορίζεται από τον προγραμματιστή στο mvote.c:13 με #define
To initial number m of buckets ορίζεται από τον προγραμματιστή στο hash_table.h:6 με #define

Δίνεται ως έξτρα mvote εντολή η print hashtable που εκτυπώνει τα περιεχόμενα του hashtable

Εξηγήσεις σχετικά με τον κώδικα δίνονται αναλυτικά στα σχόλια του προγράμματος.
Γίνεται χρήση εργαλείου Doxygen για code documentation.
Προς διευκόλυνση του εξεταστή, τα παρακάτω στοιχεία γίνονται εμφανή καθώς κάνουμε hover πάνω
ένα στοιχείο:
    @param
    @result
    @retval
    @returns
    @warning

Οδηγίες για VSCode:
    1) Προσωπικά έκανα χρήση του extension:
    https://marketplace.visualstudio.com/items?itemName=cschlosser.doxdocgen

    2) Οδηγούμαστε στο File->Preferences->Settings
    3) Πληκτρολογούμε "Doxygen" στο πεδίο Search Settings και
       βρίσκουμε το C_Cpp > Doxygen > Section Tags
    
        ή ευκολότερα Πληκτρολογούμε C_Cpp.doxygen.sectionTags 
    4) Φροντίζουμε στη λίστα των tags να υπάρχουν τα παραπάνω @
